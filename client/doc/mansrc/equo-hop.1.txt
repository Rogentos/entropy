NAME
----
equo-hop - upgrade the System to a new branch


SYNOPSIS
--------
equo hop [-h] <branch>


INTRODUCTION
------------
Upgrade the System to a new branch.



OPTIONS
-------
"equo hop" supports the following options which alters its behaviour.


POSITIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~~~
*<branch>*::
    branch

OPTIONAL ARGUMENTS
~~~~~~~~~~~~~~~~~~
*--help*::
    show this help message and exit
